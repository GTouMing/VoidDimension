# Void Dimension 虚空维度模组

一个为 Minecraft 1.21.1 设计的模组，添加了全新的虚空维度及相关游戏机制。

## ✨ 主要功能

### 虚空锚点 (Void Anchor)
- **跨维度传送**：在主世界和虚空维度间快速传送
- **遗物保存**：死亡时物品自动保存到绑定的锚点
- **能量管理**：存储和管理虚空能量
- **物品收集**：自动收集周围掉落物品
- **物品存储**：具有6*9容量的存储空间

### 虚空终端 (Void Terminal)  
- **便携控制**：远程访问和管理绑定的锚点
- **饰品栏支持**：可装备在 Curios API 饰品栏中
- **快捷键操作**：默认按 `V` 键快速打开终端界面
- **多页面界面**：信息概览、世界设置、玩家设置、锚点设置

### 遗物返还系统
- **智能保存**：死亡时保存所有物品（包括饰品栏）
- **位置记录**：返还时物品放回原槽位
- **堆栈处理**：智能合并物品堆栈

## 🚀 快速使用

### 合成配方
**虚空锚点**：
```
O E O
E H E  
O E O
```
- O = 黑曜石, E = 末影珍珠, H = 海洋之心

**虚空终端**：
```
  I R
I G I
  I 
```  
- R = 红石粉, I = 铁锭, G = 玻璃板

### 基本操作
1. **绑定锚点**：手持终端右键锚点进行绑定
2. **打开终端**：装备终端后按 `V` 键
3. **维度传送**：通过终端界面传送到绑定的锚点
4. **遗物取回**：死亡后右键锚点取回物品
